# Fashion Store E-commerce Website

A complete frontend e-commerce website for women's clothing built with HTML, CSS, Bootstrap, and JavaScript.

## Features

### 🛍️ **Product Categories**
- Summer Collection
- Winter Collection  
- Essential Wear
- Unstitched Fabrics
- Ready to Wear

### 🎨 **Design Features**
- Responsive design for all devices
- Modern UI with Bootstrap 5.3.0
- Beautiful animations and transitions
- 3D scroll effects
- Loading animations
- Toast notifications

### 🛒 **E-commerce Functionality**
- Product listing with filtering and sorting
- Product detail pages with image galleries
- Shopping cart with add/remove items
- Quantity management
- Promo code support
- Sale countdown timer

### 🎯 **Interactive Elements**
- Dynamic product data from JavaScript
- Category-wise product organization
- Size and color selection
- Star ratings
- Customer reviews
- Related products

## How to Run

### Method 1: Direct Browser Opening (Recommended)
1. Simply double-click `index.html` to open in your browser
2. No server required - everything runs locally!

### Method 2: Live Server (Optional)
If you want to use a live server for development:
1. Install VS Code Live Server extension
2. Right-click `index.html` and select "Open with Live Server"

- ❌ No server setup needed

- ❌ No Python/Node.js
